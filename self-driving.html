Overview

The "Self Driving Car Using OpenCV" project leverages computer vision techniques implemented with OpenCV and machine learning models developed using TensorFlow/Keras to enable a car to drive autonomously. The project involves collecting driving data, preprocessing it, training a deep neural network model, and deploying the model to control the car's steering and throttle in real-time. This system aims to demonstrate the fusion of computer vision and machine learning for autonomous vehicle navigation.

Key Features

Data Collection and Preprocessing

Data is collected using a simulator and stored in a CSV file containing information about image paths and corresponding steering angles. Images are preprocessed to focus on the region of interest, converted to the YUV color space, resized, and normalized.

Data Augmentation

Various data augmentation techniques such as zooming, panning, brightness alteration, and flipping are applied to increase the diversity of training data and improve model generalization.

Model Architecture

Constructs a convolutional neural network (CNN) model inspired by NVIDIA's architecture for self-driving cars. The model is trained on the preprocessed data to predict steering angles based on input images.

Training and Validation

Training includes data augmentation techniques such as zoom, pan, brightness alteration, and horizontal flipping to enhance model robustness. The model is trained on augmented data generated from the collected images. Training progress is monitored using validation data, and the mean squared error (MSE) loss function is minimized using the Adam optimizer.

Deployment

The trained model is deployed on a web server using Flask and Socket.IO. The server communicates with a front-end client, receiving telemetry data (image and speed) and sending back control commands (steering angle and throttle) to control the simulated car.

Speed Limit

A speed limit is enforced to ensure safe driving, and throttle is adjusted accordingly based on the current speed of the car.

Real-time Control

The deployed system allows real-time control of the simulated car, where the model predicts steering angles based on input images and adjusts throttle based on speed, enabling autonomous driving in the simulator.
